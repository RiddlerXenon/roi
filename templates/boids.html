Boids Algorithm - Модель стайного поведения

Алгоритм Boids (от "bird-oid objects") моделирует коллективное поведение стай птиц, косяков рыб и других групп животных.

Основные правила Рейнольдса:

1. Разделение (Separation)

a_sep = -k_sep * Σ(r_j - r_i) / |r_j - r_i|²

Избегание столкновений с соседними агентами

2. Выравнивание (Alignment)

a_align = k_align * (⟨v⟩ - v_i)

Согласование направления движения с соседями

3. Сплочение (Cohesion)

a_cohes = k_cohes * (⟨r⟩ - r_i)

Стремление к центру масс группы соседей

Уравнения движения:

v(t+1) = v(t) + a_total * dt
r(t+1) = r(t) + v(t) * dt
a_total = a_sep + a_align + a_cohes

Ограничения:

  • |v| ≤ v_max - максимальная скорость
  • |a| ≤ a_max - максимальное ускорение
  • FOV - поле зрения агента
  • r_perception - радиус восприятия

Параметры модели:

  • φ - угол поля зрения (град)
  • w_match, w_center, w_sep - веса правил
  • τ_match, τ_center, τ_sep - времена релаксации
  • r, r_sep - радиусы восприятия
